# CP1 - Programação Orientada a Objetos

## Projeto: FiapRide

Este projeto foi desenvolvido como parte da **CP1 da disciplina de Programação Orientada a Objetos**.  
A proposta da atividade consiste em revisar e ajustar um código previamente desenvolvido por um estagiário, aplicando os conceitos aprendidos nas últimas aulas.

O objetivo foi **corrigir e melhorar a estrutura do código**, garantindo uma modelagem mais adequada da classe `Veiculo`, com organização em pacotes e aplicação de boas práticas de POO, como **encapsulamento e controle de acesso aos atributos**.

## Estrutura do Projeto

O projeto foi organizado em dois pacotes:

- `br.com.fiapride.model`  
  Contém a classe `Veiculo`, responsável por representar os dados e comportamentos do veículo.

- `br.com.fiapride.main`  
  Contém a classe `SistemaPrincipal`, responsável por executar o programa e demonstrar o funcionamento da classe.

## Funcionalidades

- Registrar dono, placa e quantidade de gasolina do veículo
- Adicionar combustível
- Registrar consumo de combustível
- Exibir as informações do veículo
